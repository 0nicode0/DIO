Extra
Há mais valores junto do "=" para indicar algo!
=: Apenas herda o valor informado e ignora o resto;
~=: Identifica a palavra exata dentro do valor, ou toda a frase contendo ela;
|=: Identifica a palavra exata dentro do valor se tiver acopldo a um hífem;
^=: Procura um prefixo em que o valor vai se iniciar;
$=: Procura um sufixo em que o valor vai se iniciar;
*=: Herda tudo apenas por ter o valor indicado;

Unidade de medida VH!
A medida vh é igual a 1/100 da altura da viewport. Então, por exemplo, se a altura do navegador é 900px , 1vh equivale a 9px . E, analogamente, se a largura da viewport é 750px , 1vw equivale a 7.5px . Há infinitas possibilidades de uso com essas unidades de CSS moderno em uma folha de estilos :D

Estilizando

Seletores CSS

São as formas de se aplicar o css no código!

  • Seletor de Tags: Dentro da "head" de nosso html determinamos o estilo para todas as tags que desejarmos. Ex: h1, h2, h3 {background-color: black;}
  • Seletor de ID: Utilizamos um atributo chamado "id" que pode ser colocado do lado da tag, podendo assim nomea-lo para ficar mais fácil de achar no CSS, é bem útil quando queremos editar só uma coisa em especifico, mas que se repete não sendo uma classe. Também é editado dentro da head. Ex: #nome do id {color: green;}
  • Seletor de Classes: Identifica um grupo e aplica o CSS somente nele. Também editado na head. Ex: .nome da class {blah blah blah}
  • Seletor Universal: Basicamente pega tudo no código e aplica o CSS. Tambem na head. Ex: * (sim, é só o asterisco) {blah blah}
  • Seletor de Atributo: Aplica o CSS na tag onde se encontra o atributo com seu valor ou nome. Head. Ex: [atributo+valor] {blah}

Combinadores

Pegam seus descendentes.

  • Combinador Descendente: Pode ser aplicado com o espaço entre uma tag e outra que se deseja aplicar o mesmo estilo Css e que são descendentes. Ex: li li {background-color: black}
  • Combinador filho: É aplicado pelo sinal de maior ">" aos seus "filhos" ou a tag dentro dele. Ex: div > p {color: blue}
  • Combinador irmão adjacente: Aplicado pelo sinal de mais "+", usamos pra selecionar o elemento depois do seletor que definirmos a esquerda. Ex: div + p {color: tomato} (neste caso ele sempre pegará o primeiro "p" q apararecer, ignorando os outros).
  • Combinador irmão geral: Sinal de tíl "~", pega todos os irmãos da tag informada. Ex: div ~ p {blah blah blah}

Width e Height

Altura e largura. O width é somado da seguinte forma: width = largura + borda + padding; Neles podemos aplicar alguns elementos predefinidos:

  • auto: Aplica um valor automático, alterado sempre que o navegador achar necessário;
  • initial: Aplica o valor padrão, geralmente usado quando não se sabe esse padrão;
  • inherit: Define o valor da propriedade de acordo com seu elemento pai;
  • min e max-width: Largura máxima e minima, se adapta de acordo com o conteúdo;
  • min e max-height: Altura máxima e minima, se adapta de acordo com o conteúdo;

Margin

Cria espaçamento externo em volta do elemento. Ex: .main {margin: 10px 10px 10px 10px} (Ele aplica o valor para toda as direções podendo ser alterado). Tem suas variações tbm:

  • margin-top: Coloca margem só em cima;
  • margin-bottom: Coloca margem só em baixo;
  • margin-left: Coloca margem só na esquerda;
  • margin-right: Coloca margem só na direita.

padding

Aplica espaço interno separando o conteúdo do elemento das bordas, podendo ser aplicado para todos os lados também, as variações são as mesmas que a da margin.

box-sizing

Aceita dois valores:

  • content-box: Soma o padding com o height
  • border-box: Soma o padding com as bordas, determinando largura e altura

Propriedades de mídias

As seguintes propriedades são mais usadas para videos e imagens:

  • Object-fit: Determina dimensão de vídeo e imagem para caber em containers pre-definidos no style como width e height. Seus modos são:
    1. fill: Preenche o container distorcendo a imagem;
    2. contain: Apenas preenche o container com o tamaho padrão da imagem;
    3. cover que preenche todo o container cortando a imagem pra caber nele;
    4. none: Ignora o width e height do elemento pai mantendo seu tamanho original;
    5. scale-down:Pega o menor valor possível aplicando-o à imagem.

    Veja exemplos no playground
  • Object-position: especifica como deve ser posicionado verticalmente como horizontalmente, pode centralizar imagens ou vídeos; Ex: img {object-position: 50% 50%;}

Fundo de Elementos

  • background-color: Altera a cor do fundo;
  • background-image: Coloca uma imagem como fundo usando a função url('');
  • background-position: Muda a posição da imagem dentro do container.
  • background-attachment: Define como o fundo se comporta a algumas ações, como rolar a bolinha do mouse etc. nela pode-se definir 3 valores diferentes:
    1. fixed: Fundo fixo;
    2. scroll: Rola junto com a div principal;
    3. local: Imagem rola junto com o conteúdo do elemento

    Veja exemplos no playground

    • Background-origin

    Define ponto de origem do fundo. Também tem sues valores:

    1. Padding-box: Valor padrão, canto superior esquerdo;
    2. border-box: Da esquerda cobrindo espaçamento externo;
    3. content-box: Se posiona da esquerda, mas não cobre a borda nem o espaçamento externo.

    Background-clip

    Permite controlar a imagem ou cor de fundo, preenche a área de borda, a área externa ou o conteúdo. Seus valores são iguais o do origin.


    Background-blend-mode

    Define como a imagem de fundo ou cor se misturam. Seus modos de mesclagem são muitos, tente brincar com eles no playground e ver o que cada um faz (Explicar vai demorar muito...)


Bordas

  • border: A mais comumente usada. É possível alterar seu estilo e cor;
  • border-radius: Arredonda cantos de objetos com px e %;
  • border-image: Coloca uma imagem como borda. Tem suas variações
  • border-style: Define estilo da borda;

Fontes

Existem 3 formas de mudar a fonte do seu código:

  1. font-family: Pode ser usado dentro do body mesmo do html, dai é só alterar a fonte a fonte;
  2. @font-face: Altera a fonte dentro do arquivo .css;
  3. @import url(): Importa uma fonte especifica a partir de uma url aplicando configurações predefinidas do font-face.

Algumas variações de fontes:

  • font-weight: Muda o "peso" da fonte;
  • font-style: Muda o estilo da fonte;
  • font-variant: Vareia no formato e tamanho da fonte. Só aceita dois valores: small-caps que diminui o formato e tamanho e o normal, q deixa no tamanho e formato padrão;
  • font-stetch: Deixa o texto mais estreito;
  • line-height: Define espaçamento em cima e embaixo do texto;

Se quiser saber mais pesquise e brinque no playground!

Text

Vamos ver algumas propriedades de estilização de texto (de novo).

  • text-transform: Define caracteres maiúsculos e minusculos, existem muitas variações... Vai brincar no playground.
  • text-align: Alinha o texto.
  • text-decoration: Adiciona ou remote linhas do texto, aceita alguns valores. Vai brinca!
  • text-ident: Define recuo da primeira linha do nosso texto;
  • letter-spacing: Adiciona espaçamento entre as letras;
  • word-spacing: Adiciona espaço entre as palavras;
  • white-space: Define como espaço dos textos se comportam, aceita vários valores, é só pesquisar e brincar com eles no playground pra saber o que cada um faz
  • word-wrap: Quebra a palavra para que caiba no container;
  • word-break: Quebra a palavra caso seja muito grande;
  • writing-mode: Define a orientação de leitura do texto;
  • text-overflow: Define como vamos mostrar que há mais conteúdo no texto além do que se pode visualizar naquele momento;

Shadow

Bora adicionar profundidade!

  • box-shadow: Adiciona uma caixa de sombra atrás do objeto;
  • filter: Adiciona alguns "filtros", sendo que drop-shadow é um de seus valores. Sim, ele adiciona sombra também.
  • text-shadow: Adiciona sombra em texto;

Outras propriedades...

  • opacity: Adiciona trasparencia;
  • overflow: Define como o conteúdo deve ser exibido caso ultrapasse os limites do container. Tem alguns valores QUE VOU LISTAR DEPOIS;
